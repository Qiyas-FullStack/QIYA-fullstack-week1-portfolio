# 🚀 QIYA Full-Stack Training: Week 1 Assignment
## Project: Building Your Personal Portfolio

Welcome to the first week of your QIYA Full-Stack Development journey! This week, you will build the foundation of your professional online presence by creating a personal portfolio page.

## 🎯 Assignment Goals
- Build a responsive personal profile page using **HTML, CSS and JavaScript**.
- Master basic **Git and GitHub** workflows.
- Learn to use **Branching** for feature development.


## 🛠 Step-by-Step Instructions

### 1. Clone the Repository
First, clone this repository to your local machine:
```bash
git clone https://github.com
cd QIYA-fullstack-week1-portfolio
```

### 2. Create a Feature Branch
To keep the `main` branch clean, create a new branch named after yourself:
```bash
git checkout -b feature/your-name-portfolio
```

### 3. Build Your Portfolio
Create a folder named `portfolio` and add your project files:
- `index.html`: Your main profile content.
- `style.css`: Your custom styling and layout.
- `/assets`: A folder for your profile picture and other images.

### 4. Commit and Push Your Changes
Save your progress frequently with clear commit messages:
```bash
git add .
git commit -m "Initial commit: Add bio and contact section"
git push origin feature/your-name-portfolio
```
