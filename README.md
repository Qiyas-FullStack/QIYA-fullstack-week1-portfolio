# QIYA Full-Stack Week 1: Personal Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript as part of the QIYA Full-Stack Development training program.

## 🚀 Features

- **Responsive Design**: Mobile-first approach with modern CSS Grid and Flexbox
- **Dark Theme**: Professional dark mode with gradient accents
- **Interactive Elements**: Smooth scrolling, cursor effects, and project modals
- **Sections**: Hero, About, Skills, Projects, Experience, Contact
- **Modern UI**: Glassmorphism effects, animations, and custom typography

## 🛠 Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Custom properties, gradients, animations, and responsive design
- **JavaScript (ES6+)**: DOM manipulation, event handling, and dynamic content
- **Font Awesome**: Icons for visual enhancement
- **Google Fonts**: Inter and JetBrains Mono for typography

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Qiyas-FullStack/QIYA-fullstack-week1-portfolio.git
   cd QIYA-fullstack-week1-portfolio
   ```

2. Open the portfolio:
   - Navigate to the `portfolio` folder
   - Open `index.html` in your web browser
   - Or use a local server (recommended for full functionality)

### Using a Local Server
For the best experience with all features working:
```bash
# Using Python (if installed)
python -m http.server 8000

# Or using Node.js (if installed)
npx serve portfolio

# Then open http://localhost:8000 in your browser
```
